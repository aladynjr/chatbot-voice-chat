Voice Chat Assistant Setup

To enable voice chat features, you'll need an ElevenLabs API key. This powers the natural-sounding voice responses in our extension.
Note: ElevenLabs offers a free tier that includes 10,000 characters per month - perfect for trying out the voice chat feature.

How to get an API key:

  1. Visit elevenlabs.io
  2. Create an account or sign in (includes free trial)
  3. Click your profile icon in the top right
  4. Select "Profile"
  5. Find your API key in the settings
  6. Copy and paste it here
