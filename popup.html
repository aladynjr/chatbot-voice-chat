ElevenLabs API Key

We use ElevenLabs' AI technology to generate natural-sounding voice responses for the chat. You'll need an API key to enable this feature.

How to get your API key:

  1. Go to elevenlabs.io
  2. Create an account or sign in (includes free trial)
  3. Click on your profile icon in the top right
  4. Select "Profile"
  5. Scroll down to "API Key"
  6. Copy and paste your key here
